# Pranab Debnath Portfolio Website

A modern, responsive single-page Data Analytics portfolio.

## Structure
- `index.html`: Entry point, Tailwind config
- `app.js`: Main layout (Sidebar + Main Content area)
- `components/`:
  - `Sidebar.js`: Fixed profile card
  - `Projects.js`: Fetches GitHub repos via API
  - `Analytics.js`: Shows Github chart and stats
  - `CompetitiveProgramming.js`: HackerRank & LeetCode stats
  - `Experience.js`: Work history
  - `Education.js`: Education details
  - `Contact.js`: Contact form with mailto integration

## Features
- Dynamic GitHub project fetching
- Mobile responsive (Sidebar becomes top header)
- Smooth scrolling navigation
- Integrated contact form with encrypted phone number protection
