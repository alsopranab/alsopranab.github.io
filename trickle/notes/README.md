# Data Analyst & Tech Professional Portfolio

## Overview
This is a single-page portfolio website designed for a Data Analyst & Tech Professional. It features a modern, high-tech dark theme with neon accents.

## Sections
- **Hero**: Introduction with futuristic visuals.
- **About**: Bio and skills matrix.
- **Projects**: Grid of data projects.
- **GitHub Activity**: Dashboard-style widgets for GitHub stats.
- **Analytics Dashboard**: A visual representation of professional metrics using charts.
- **Contact**: Functional contact form UI.

## Tech Stack
- React 18
- TailwindCSS
- Chart.js (for analytics visualization)
- Prism.js (for syntax highlighting)
- Lucide Icons

## New Features
- **Code Viewer**: Integrated VS Code-style code blocks with syntax highlighting for project previews.
- **Lightbox**: Image preview modal for certifications.
- **Cyber Grid**: Animated background effect in the Hero section.
